Container Tags

Container Tags - Which don't have their own content. Examples like
  • table
  • body
  • head
  • div

Standalone Tags

Standalone Tags - Which need not be closed. Examples
  • br
  • img
  • hr

Content Tags

Content Tags - Which must be opened and closed. The content is entered in between opening and closing tag. Examples
  • p
  • b
  • i
  • h1.....h6
